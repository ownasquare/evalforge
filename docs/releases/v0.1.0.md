# EvalForge v0.1.0

EvalForge v0.1.0 is a local-first public beta for comparing prompts and models against the same
test set before shipping an LLM feature.

## Core workflow

1. Choose or import a benchmark.
2. Select prompt and model candidates against a baseline.
3. Run the evaluation and review correctness, relevance, groundedness, hallucination risk,
   constraints, latency, and known estimated cost.
4. Compare candidates and export reviewable, content-redacted evidence.

The deterministic demo is key-free and works offline:

```bash
uv sync --frozen
uv run evalforge demo
```

The release includes the FastAPI service, Streamlit dashboard, SQLite local mode, PostgreSQL
contracts for shared workers, explainable built-in metrics, source-level extension examples, and
versioned evidence exports.

## Release boundaries

This release does not certify hosted OIDC, paid-provider calibration, production deployment, PyPI
publication, or container-registry publication. Those require environment-specific identity,
data-transfer, spend, infrastructure, backup, monitoring, and readback proof. EvalForge v0.1.0 is
not published to PyPI or a container registry; the GitHub Release provides the source distribution,
wheel, and SHA-256 checksums.
